     ./scripts/run-tests.sh quick

     Use: Before commits, quick validation

  2. Integration Tests (2-5 minutes)
     ./scripts/run-tests.sh integration
     Profile: -Pintegration-tests
     Use: Test with real services (requires Docker)

  3. Full Test Suite (10-15 minutes)
     ./scripts/run-tests.sh all
     ./scripts/run-tests.sh full
     Profile: -Pfull-test-suite
     Use: Before pull requests, complete validation

     ./scripts/run-tests.sh coverage
     Action: Generates and opens JaCoCo report in browser
     Use: Check test coverage details

     ./scripts/run-tests.sh --help
     ./scripts/run-tests.sh help
     ./scripts/run-tests.sh -h
     Action: Display help text with all options

  6. Clean Artifacts
     ./scripts/run-tests.sh clean
     Action: Remove test artifacts and target directory
     Use: Fresh rebuild
