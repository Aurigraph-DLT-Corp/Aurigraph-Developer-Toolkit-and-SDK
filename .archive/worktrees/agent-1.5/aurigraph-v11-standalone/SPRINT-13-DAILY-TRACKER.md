## üÜò BLOCKER ESCALATION

### If You're Blocked

1. **Acknowledge it in standup** (10:30 AM)
2. **Describe the blocker** (be specific)
3. **Ask for help** (who can unblock you?)
4. **Timeline** (how long can you wait?)

### Escalation Path

| Time | Action | Contact |
|------|--------|---------|
| Immediately (if blocking others) | Flag in standup | Frontend Lead |
| < 1 hour | Pair programming | Relevant expert |
| < 4 hours | Architecture review | PM + Frontend Lead |
| > 4 hours | Escalate to PM | Project Manager |

### Common Blockers & Solutions

**"Mock API not returning expected data"**
- Check: MOCK-API-SERVER-SETUP-GUIDE.md
- Escalate: DevOps Lead

**"Component performance target not met"**
- Check: SPRINT-13-15-PERFORMANCE-BENCHMARKS.md
- Escalate: QA Lead

**"Test coverage not at 85%"**
- Check: Test examples in codebase
- Escalate: QA Lead

**"Git merge conflicts"**
- Fetch latest, resolve conflicts locally
- Ask for help: Any senior developer

**"TypeScript compilation error"**
- Check eslint config
- Ask: Frontend Lead
