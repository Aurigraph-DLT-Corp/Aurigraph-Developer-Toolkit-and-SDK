## Production Notes

### Screen Recording Requirements

**Resolution:** 4K (3840x2160) downscaled to 1080p
**Frame Rate:** 60fps for smooth UI animations
**Browser:** Chrome with no extensions
**Zoom Level:** 90% for full dashboard visibility

### Pages to Record

1. **Home/Dashboard** - `https://dlt.aurigraph.io/`
2. **Demo Page** - `https://dlt.aurigraph.io/demo`
3. **Marketplace** - `https://dlt.aurigraph.io/marketplace`
4. **Asset Registry** - `https://dlt.aurigraph.io/rwa/registry-navigation`
5. **Tokenize Asset** - `https://dlt.aurigraph.io/rwa/tokenize`
6. **Portfolio** - `https://dlt.aurigraph.io/rwa/portfolio`
