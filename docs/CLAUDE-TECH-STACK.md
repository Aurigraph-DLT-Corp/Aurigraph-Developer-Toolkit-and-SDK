## Tech Stack Details

### Backend
- **Framework**: Quarkus 3.x
- **Language**: Java 21
- **Communication**: gRPC, REST
- **Database**: LevelDB, PostgreSQL
- **Build**: Maven

### Frontend
- **Framework**: React 18
- **Language**: TypeScript
- **Build Tool**: Vite
- **UI Kit**: Ant Design
- **Management**: npm

### Testing
- **E2E**: Playwright
- **Unit**: JUnit 5, Mockito
