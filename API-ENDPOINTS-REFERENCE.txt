CRITICAL CONFIGURATION ISSUES
================================================================================

ISSUE #1: Hardcoded Localhost URLs (Production Risk)
  Files:
    - ExternalAPITokenization.tsx    Uses http://localhost:9003/api/v11/tokenization/*
    - QuantumSecurityPanel.tsx       Uses http://localhost:9003/api/v11/security/scan
  Impact: Will fail in production unless manually configured
  Solution: Use API_BASE_URL instead

ISSUE #2: Hardcoded Port 8443 (Incorrect Configuration)
  File: RicardianContractUpload.tsx
  Endpoint: https://dlt.aurigraph.io:8443/api/v11/contracts/ricardian/*
  Problem: Port 8443 doesn't match main API (standard HTTPS uses 443)
  Solution: Use same base URL as other endpoints

ISSUE #3: Hardcoded Demo URLs
  File: MerkleRegistryViewer.tsx
  Endpoint: https://dlt.aurigraph.io/api/v11/demo/registry
  Status: Demo-only, acceptable

ISSUE #4: ComprehensivePortalService Hardcoded Base URL
  File: ComprehensivePortalService.ts (line 28)
  Current: constructor(baseUrl: string = 'http://localhost:9003', demoMode: boolean = false)
  Should: Use API_BASE_URL from constants
